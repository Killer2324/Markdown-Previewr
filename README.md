# Markdown Previewr

## Contributions

1. Fork this repository
2. Clone repository

**SSH**

```bash
git clone git@github.com:YourUser/Markdown-Previewr.git
```

**HTTPS**

```
git clone https://github.com/YourUser/Markdown-Previewr.git
```

3. Create a new branch
4. Make your changes
5. Create a pull request
